# Getting Started

Welcome to F5 Distributed Cloud Tools for VS Code! This section will help you
get up and running quickly.

## In This Section

- [Installation](installation.md) - How to install the extension
- [Quick Start](quick-start.md) - Your first steps with F5 XC Tools
- [Authentication](authentication.md) - Setting up your credentials

## Prerequisites

Before you begin, ensure you have:

- VS Code 1.85.0 or higher installed
- Access to an F5 Distributed Cloud tenant
- Valid F5 XC credentials (API Token or P12 certificate)
